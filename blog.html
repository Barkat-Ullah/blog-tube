Discuss the scope of var, let, and const.
Var:
  1. It has used in Global or Function scoped
  2. variables can be updated and re-declared within its scope
Let:
  1. Block scoped
  2. let can be updated but not re-declared
Const:
  1. It's Block scoped
  2. const variables can neither be updated nor re-declared.
Tell us the use cases of null and undefined?

Undefined : Undefined means the variable has been declared, but its value has not been assigned.

Null : Null means an empty value or a blank value.
